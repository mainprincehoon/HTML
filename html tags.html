1. <html>: The root element that wraps all the content of an HTML document.
  2. <head>: Contains metadata about the document, such as the title and links to stylesheets or scripts.
  3. <title>: Specifies the title of the document, displayed on the browser tab.
  4. <body>: Contains the main content of the document, visible to users in the browser.
  5. <h1> to <h6>: Headings used to define titles or subtitles, where <h1> is the largest and <h6> is the smallest.
  6. <p>: Represents a paragraph of text.
  7. <a>: Defines a hyperlink, used to link to other pages, sections, or resources.
  8. <img>: Embeds an image into the document, requiring a "src" attribute for the source file.
  9. <ul>: Defines an unordered list, typically with bullet points.
  10. <ol>: Defines an ordered list, with items numbered or lettered.
  11. <li>: Represents an individual list item within <ul> or <ol>.
  12. <div>: A container element used to group content or elements for styling or layout purposes.
  13. <span>: An inline container for styling or grouping text without affecting its flow.
  14. <form>: Creates a form to collect user input, often used with various input fields.
  15. <input>: A versatile element for capturing user input, such as text, numbers, checkboxes, or radio buttons.
  16. <button>: Defines a clickable button for submitting forms or triggering actions.
  17. <table>: Used to create a table structure, often paired with <tr>, <th>, and <td> tags.
  18. <tr>: Represents a row in a table.
  19. <th>: Defines a header cell in a table.
  20. <td>: Represents a standard data cell in a table.
  21. <header>: A semantic element for defining introductory content or navigation links.
  22. <footer>: A semantic element for footer content, such as copyright or contact information.
  23. <nav>: Represents a section of navigation links.
  24. <section>: Groups related content into a thematic section.
  25. <article>: Defines standalone content, such as a blog post or news article.
  26. <aside>: Represents secondary or complementary content, often used for sidebars.
  27. <video>: Embeds a video file, with attributes for controls, autoplay, and more.
  28. <audio>: Embeds an audio file, with attributes for controls and autoplay.
  29. <canvas>: Provides a drawing surface for graphics or animations using JavaScript.
